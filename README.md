# NHOM-3-TUAN-9

Mở rộng các chức năng:
- Mở rộng vị trí: tầng - kệ - kho.
- Tìm kiếm theo tác giả và tựa đề.
- Sắp xếp theo giá bán: tăng / giảm.
- Tính tổng giá trị kho.
- Lưu / Nạp CSV.
- Xử lý ngoại lệ toàn hệ thống.
- Update (cập nhật) dạng đa hình:
  - Sách giáo trình cập nhật được: môn học, cấp độ.
  - Sách tiểu thuyết cập nhật: thể loại, series.
  - Các trường chung (tựa đề, tác giả, số lượng, giá…) được cập nhật cho cả hai loại.
- Kiểm tra dữ liệu đầu vào:
  - Mã sách: chỉ chữ + số.
  - Tựa đề: chữ + số + khoảng trắng.
  - Tác giả: chỉ chữ.
  - Năm xuất bản: 1 – 2025.
  - Số lượng ≥ 0.
  - Giá cơ bản ≥ 0.
  - Môn học: chữ + số.
  - Cấp độ: đại học / phổ thông.
  - Vị trí phải chứa: tầng / kệ / kho.
